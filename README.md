# Sol Tutoring And Coding Academy

Full-stack website for local tutoring business.

## Live Demo
- Frontend: https://stca-academy.netlify.app
- Backend: https://sol-tutoring-api.onrender.com

## Features
- Mobile responsive with hover effects
- Contact form with MongoDB storage
- Newsletter subscription
- Admin dashboard
- 6 services: Math, Science, Computing, Exam Prep, Game Dev, AI/ML

## Tech Stack
- Node.js + Express + MongoDB
- HTML5, CSS3, JavaScript
- Deployed on Render + Netlify

npm install
npm run dev
